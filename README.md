**Proton Technosystem** is a cutting-edge single-page application (SPA) designed for a premier electronics repair service. It combines **premium aesthetics** with **functional utility**, offering users a seamless experience to explore services, get quotes, and book repairs.

## ğŸš€ Getting Started

### Installation

1.  **Clone the repo**
    ```bash
    git clone https://github.com/x-neon-nexus/Proton-Technosystem.git
    ```
2.  **Install packages**
    ```bash
    npm install
    ```
3.  **Run locally**
    ```bash
    npm run dev
    ```

### Deployment

To deploy to GitHub Pages:

```bash
npm run deploy
```
