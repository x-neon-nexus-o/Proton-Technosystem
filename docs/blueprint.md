# **App Name**: Proton Technosystem

## Core Features:

- Display workshop details: Displays workshop and services details, to bring the company vision in an accessible format.
- Lead generation: Gathers the information that is provided by the user (Name, Phone, Email, Message), so a quotation can be created. User also has a 'Quick Contact' option.
- Service Description: Presents an organized catalog of services (Projector Repair, Camera Repair, etc.) via interactive flip cards and other navigation aids.

## Style Guidelines:

- Primary color: Deep Crimson (#e60023) for a trustworthy, expert-driven feel.
- Secondary color: Proton Blue (#0057b7).
- Accent color: Jet Black (#0d0d0d).
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look.
- Use minimal, white outline icons that glow on hover, paired with small photo thumbnails in service descriptions.
- Incorporate smooth fade-ins, gentle hover glows on interactive elements, and a parallax effect for the hero section.
